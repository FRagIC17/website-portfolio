Projects

Here are some of the projects I have worked on:

Miniproject 1

This is a guessing game where the program picks a random number between 0 and 100, and the user has to guess it. It provides hints like "Way off!" or "Getting close!" based on how near the guess is, and ends when the number is guessed correctly or after 10 attempts.

Miniproject 2

This is a program that checks whether an entered email is valid. It verifies if the email has the correct length, contains exactly one "@" symbol, and ends with ".dk". Additionally, it ensures the email meets specific formatting requirements to be considered valid for use.

Miniproject 3

This is a password validation program that checks if a password meets specific security requirements. It ensures the password is at least 8 characters long, contains both uppercase and lowercase letters, includes at least one digit, and has at least one special character. If the password fails any of these checks, it provides feedback on what is missing.

grundforløbsproject

This is a GUI-based program with two features: a BMI calculator and a working hours calculator. Using PySimpleGUI, it allows users to input their height and weight for BMI calculation or enter start and end times to compute working hours. It includes error handling and displays results interactively.
